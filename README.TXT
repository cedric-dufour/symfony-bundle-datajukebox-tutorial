Data Jukebox Tutorial Bundle
============================


Synopsis
--------

The Data Jukebox Tutorial Bundle demonstrates the usage of the Data Jukebox
Bundle via a sample blog application, along detailed documentation and
explanations.
 

Dependencies
------------

 - [MUST] Symfony 2.7 or later
 - [MUST] PHP 5.3.9 or later
 - [SHOULD] Data Jukebox Bundle
